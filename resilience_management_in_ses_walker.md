SES management assumes
1. probabilistic responses to management
2. predictable changes in climate and technology

Use manager as adaptive part of the system.
Uncertainties as large as system changes (irreducible)
Look at coarse-grained features
Holling 1992, adaptive cycle
Decision analysis evaluates probability distribution of proximate options. Issues:
1. Probabilities unknown
2. No single utility/loss function
3. No utility function constructued yet
4. Reactions to future forecasts
5. Model representation deviates from system

Resilience analysis consciously weights for outliers/catastrophes
1. Thresholds with hysteretic, irreversible changes are NOT CONVEX
2. Uncertain variable distributions, including improper priors
3. Decisions based on imperfect knowledge
4. Market distortions, valuations are normal
5. Agents hold preferences over processes, not just outcomes (legitimacy)
6. Well-defined property, markets for resources do not exist
7. System state: collection of state variables
8. Configuration = collection of states (eg. grassland)

1. Amount of stress to retain ctrls of function and structure (within same domain of attraction)
2. Self-organization (complex)
3. Capacity for learning/adaptation (adaptive)

a = renewal, reorganization
